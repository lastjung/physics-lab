# DoD Checklist

## Common
- [ ] 요구한 기능이 코드에 실제 반영됨
- [ ] 관련 테스트가 추가되었거나 기존 테스트로 증명됨
- [ ] `pnpm test` 통과
- [ ] `pnpm build` 성공
- [ ] 문서 상태(`JAEMIN_TASKS.md`)가 코드와 일치

## New Feature
- [ ] 입력/경계조건 처리
- [ ] 회귀 리스크 포인트 테스트
- [ ] 사용자 조작 경로(최소 1개) 확인

## Bug Fix
- [ ] 재현 조건 명시
- [ ] 실패 케이스 -> 수정 후 통과 케이스 확인
- [ ] 부작용 여부 점검
